Command, CMD

CMD is a networking website that I gave my modern and artistic take on. Using gradients and effective use of dark space, I was able to design, using AdobeXD, the home & login pages for both desktop and mobile use. This is one of my first projects in AdobeXD, after years of experience in photoshop, this is how I learned the basics of XD- for learning the basics, I think I did pretty damn well.

ParkMark

ParkMark is a solo project that I, myself, curated UI/UX for entirely. This is an interactive map app who's target audience is skaters, just like me. This map app allows skaters to interact with eachother and create virtual skateparks around the world using geolocation tags. This is not an implemented app as of yet, but I have plans to do such.
